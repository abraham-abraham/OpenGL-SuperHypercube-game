

CONTROL KEYS:
____________

- left arrow : counter clockwise world rotation around +x-axis
- right arrow : counter clockwise world rotation around -x-axis
- up arrow : counter clockwise world rotation around +y-axis
- down arrow : counter clockwise world rotation around -y-axis
- Home : reset to initial world orientation


- Space : move model back to initial position

- "U" : scale up (must be "shift" + "U")
- "J" : scale down (must be "shift" + "J")

Camera:

"Shift" + any of the following camera movements : increases the speed of the camera x2

- ";" : move camera to the left
- "Enter" : move camera to the right
- "[" : move camera up (the y-axis)
- " ' " (apostrophe): move camera down (the y-axis)
- "End" : move camera down the z-axis
- "Page Up" : move camera up the z-axis


- Left mouse button + mouse movement : zoom 
- Right mouse button + mouse movement : pan camera (horizontal movement)
- Middle mouse button + mouse movement : tilt camera (vertical movement)




Changes to the world:

- "X" : toggle texture on/off
- "B" : toggle shadows on/off

- "P" : render points
- "L" : render lines
- "T" : render triangles
